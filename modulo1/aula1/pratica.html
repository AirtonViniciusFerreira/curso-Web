Aula 1 - Prática

No HTML os arquivos tem que ter a extensão ".html" no nome do arquivo. O mais comum dos arquivos Web é o arquivo index.html que é a página inicial do site.

Na prática o mais usado é usar extensão na IDE do inglês Integrated Development Environment ou Ambiente de Desenvolvimento de programação para facilitar a escrita dos elementos HTML. Todo elemento precisa ser aberto <elemento> e fechado </elemento>

Para criar uma paágina HTML, basta criar um arquivo ".html" e colocar a estrutura base do HTML. Como mostrado neste exemplo que mostra um Olá mundo. 

                        <html>
                            <head>

                            </head>
                            <body>
                                Olá Mundo
                            </body>
                        </html>

Assim a página irá ser aberta, porém será aberta como um arquivo normal. Tem algumas coisas que só funcionará se a página for servida, mas não precisará de um servidor na internet sempre para ver a página. Pode servir localmente como por exemplo usando a extensão "live server" no Visual Studio Code. Com esta extensão pode mudar o código e visualizar apenas salvando o arquivo, assim pode programar dividindo tela ou usando outra tela e ir vendo o resultado na hora.

Este exemplo de cima não tem nenhuma formatação, então temos que usar o estilo com o elemento <style></style>. Aqui está um exemplo mudando a cor do texto de todos os elementos "h1" que é um cabeçalho. Estamos mudando de preto para vermelho: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>
                                <h1>Olá mundo</h1>
                                Olá Mundo
                            </body>
                        </html>

Temos diversos elementos no HTML, tem o link onde aponta o endereço de uma outra página do seu site ou de outros sites. Para isso utilizamos o elemento "<a href="https://www.uenp.edu.br" >Ir para uenp</a>", entre o elemento colocamos o texto que irá aparecer para clicar, como por exemplo, o nome do site ou o nome da página, e dentro do primeiro elemento usamos o href=" " e dentro das aspas colocamos o endereço da página. Como no exemplo a seguir: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>
                                <h1>Olá mundo</h1>
                                Olá mundo
                                <a href="https://www.uenp.edu.br">Ir para uenp</a>
                            </body>
                        </html>

Podemos ter também o JavaScript dentro dos arquivo HTML, para tornar a página dinâmica. Colocamos o elemento <script></script> dentro do elemento head, como no exemplo a seguir: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                                <script>
                                    function foo(){
                                        alert("Olá mundo");
                                    }
                                </script>
                            </head>
                            <body>
                                <h1>Olá mundo</h1>
                                <h2>Bem-vindo ao curso</h2>
                                <button onclick="foo()">Cliqui-me</button>
                                <a href="https://www.uenp.edu.br">Ir para uenp</a>
                            </body>
                        </html>

Ou colocamos o JavaScript no final do HTML para que carregue todos os elementos e depois execute o código JavaScript que pode demorar alguns segundos a mais. 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>
                                <h1>Olá mundo</h1>
                                <h2>Bem-vindo ao curso</h2>
                                <button onclick="foo()">Cliqui-me</button>
                                <a href="https://www.uenp.edu.br">Ir para uenp</a>
                            </body>
                            <script>
                                function foo(){
                                    alert("Olá mundo");
                                }
                            </script>
                        </html>
