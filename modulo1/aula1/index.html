Aula 1 - Teoria

Nesta aula teremos uma visão geral da internet, arquitetura e tecnologias para o desenvolvimento web.



A Internet é a rede mundial de computadores interligados, que permite a troca de informações utilizando o protocolo TCP/IP. Consiste de milhões de empresas privadas, públicas, acadêmicas e de governo, com alcance local e global e que está ligada por uma ampla variedade de tecnologias de rede eletrônica, sem fio e ópticas.


Hoje em dia todo mundo usa a internet o tempo todo para fazer compras, para jogos, para pesquisas e para assitir filmes e videos.



A internet é uma arquitetura cliente-servidor onde tem um servidor que pode entregar para diversos clientes, sendo eles computador, nootebook, celular, entre outros. como mostra a foto a seguir:

O computador utiliza um browser para fazer uma requisição onde vai para a internet e para o servidor web. O servidor analisa e vê os dados necessários para aquela requisição e se precisar busca no banco de dados e depois devolve para o computador uma resposta sendo a pagina HTML através do HTTP. HTML

No início o HTML era utilizado para a vizualização de artigos e por isso apenas o HTML consegue estruturar artigos.

O HTML siginifica Hyper Text Markup Language, ou no português Linguagem de Marcação de Hipertexto. Os elementos do HTML informa ao navegador como exibir o conteúdo, como por exemplo o elemento "p" que informa um parágrafo, e o "h1" que define um cabeçalho grande.


A parte da configuração e metadados estará dentro do head, e tudo que é visivel estará dentro do body. Este é um exemplo básico de como funciona o HTML: 

                        <html>
                        <head> 
                            <title> Titúlo do Documento </title>
                        </head>
                        <body> 
                            <h1> Este é um cabeçalho </h1>
                            <p> Este é um parágrafo </p>
                        </body>
                        </html>


    Elementos mais comuns:
  • <!DOCTYPE> html que é a declaração de um domumento HTML5
  • <head> elemento da meta informações sobre a página
  • <body> elemento que define o corpo do documento, todo conteúdo visível
  • <title> elemento mostra o título da página HTML
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> elementos de cabeçalho de diferentes tamanhos
  • <p> elemento que define um parágrafo

para mais informações sobre HTML consultar: w3scools



CSS

O CSS é uma ferramenta na qual consegue-se definir estilos para sua página. Com ele se faz formatação de fonte, imagens, bordas, e até mesmo animações. Ele garante também o designer responsivo para que seja utilizado em smartphones.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags <style></style>. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos

Este é um exemplo de CSS: 

                        body{
                            background-color: lightblue;
                        }

                        h1{
                            color: white;
                            text-align: center;
                        }

                        p{
                            font-family: verdana;
                            font-size:20px;
                        }

para mais informações sobre CSS consultar: w3scools



JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

JavaScript permite a criação de páginas dinâmicas com a interação do usuário e pode ser usada nos navegadores ou diretamente no sistema operacional(NodeJS).

Assim como o CSS, o JavaScript pode ser escrito diretamente na página HTML com o elemento <script></script>, ou também criar um arquivo separadamente com a extensão ".js" com todo o código e depois importar para HTML. 

                        <!DOCTYPE>
                        <html>
                        <head>
                            <title> Título do Documento </title>
                            <script>
                                function myFunction(){
                                    document.getElementById("demo").innerHTML = "parágrafo Alterado.";
                                }
                            <script>
                        </head>
                        <body> 
                            <h1> Este é um cabeçalho </h1>
                            <p id="demo"> Um parágrafo </p>
                            <button type="button" onclick="myFunction()">Tente</button>
                        </body>
                        </html>

para mais informações sobre JavaScript consultar: w3scools
