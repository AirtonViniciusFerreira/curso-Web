Aula 1 - Pratica

No HTML os arquivos tem que ter a extensão .html no nome do arquivo. O mais comum dos arquivos Web é o arquivo index.html que é a pagina inicial do site.

Na pratica o mais usado é usar extensão na ide de programação para facilitar a escrita dos elementos HTML. todo elemento precisa ser aberto assim, <elemento> e fechada assim, </elemento>

Para criar uma pagina HTML, basta criar um arquivo .html e colocar a estrutura base do HTML. Como mostrado neste exemplo que mostra um Ola mundo. 

                        <html>
                            <head>

                            </head>
                            <body>
                                Olá Mundo
                            </body>
                        </html>

Assim a pagina ira ser aberta, porem sera aberta como um arquivo normal. Tem algumas coisas que so funcionara se a pagina for servida, mas não precisara de um servidor na internet sempre para ver a pagina. Pode servir localmente como por exemplo usando a extensão live server no Visual Studio Code.

Com esta extensão pode mudar o codigo e ja visualizar apenas salvando o arquivo, assim pode programar dividindo tela ou usando outra tela e ir vendo o resultado na hora.

Este exemplo de cima não tem nenhuma formatação ai temos que usar o estilo com o elemento <style></style>. Aqui esta um exemplo mudando a cor do texto de todos elementos h1,que é um cabeçalho. Estamos mudando de preto para vermelho: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>
                                <h1>Ola mundo</h1>
                                Olá Mundo
                            </body>
                        </html>

Temos diversos elementos no HTML, tem o link onde aponta o endereço de uma outra pagina do seu site ou de outros sites também. Para isso utilimos o elemento <a href="https://wwww.uenp.edu.br" >Ir para uenp</a> entre o elemento colocamos o texto que ira aparecer para clicar como por exemplo o nome do site ou o nome da pagina. E dentro do primeiro elemento usamos o href="" e dentro das aspas colocamos o endereço da pagina. Como no exemplo a seguir: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>
                                <h1>Ola mundo</h1>
                                Olá Mundo
                                <a href="https://wwww.uenp.edu.br">Ir para uenp</a>
                            </body>
                        </html>

Podemos ter também o JavaScript dentro dos arquivo HTML, para torna a pagina dinamica. Colocamos o elemento <script></script> dentro do elemento head, como no exemplo a seguir: 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                                <script>
                                    function foo(){
                                        alert("Ola mundo");
                                    }
                                </script>
                            </head>
                            <body>
                                <h1>Ola mundo</h1>
                                <h2>Bem vindo ao curso</h2>
                                <button onclick="foo()">Cliqui-me</button>
                                <a href="https://wwww.uenp.edu.br">Ir para uenp</a>
                            </body>
                        </html>

Ou colocamos o JavaScript no final do html para que caregue todos os elementos e depois execute o codigo JavaScript que pode demorar alguns segundos a mais. 

                        <html>
                            <head>
                                <style>
                                    h1{
                                        color: red;
                                    }
                                </style>
                            </head>
                            <body>

                                <button onclick="foo()">Cliqui-me</button>
                                <a href="https://wwww.uenp.edu.br">Ir para uenp</a>
                            </body>
                            <script>
                                function foo(){
                                    alert("Ola mundo");
                                }
                            </script>
                        </html>
