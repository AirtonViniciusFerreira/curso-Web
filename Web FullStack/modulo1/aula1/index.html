Aula 1 - Teoria

Nesta aula Teremos uma visão geral da Internet, arquitetura e Tecnologias para o desenvolvimento web.



A Internet é rede mundial que interligadas, que permite a troca de informações utilizando o protocolo TCP/IP. consiste de milhões de empresas privadas, públicas, acadêmicas e de governo, com alcance local e global e que está ligada por uma ampla variedade de tecnologias de rede eletrônica, sem fio e ópticas.


Hoje em dia todo mundo usa a Internet o tempo todo, para fazer compras, para jogar, para pesquisas, para assitir seus filmes e videos.



A internet é uma arquitetura cliente-servidor. Onde tem um servidor que pode entregar para diversos clientes, sendo eles computador, nootebook, celular, entre outros. como mostra na foto a segir:

O computador utiliza um browser para fazer uma requisção onde vai para internet, e vai para o servidor web. Onde o servidor analisa e ver os dados necessarios para aquela requisição e se precisar busca no banco de dados e depois devolve para o computador uma resposta sendo a pagina HTML. atraves do HTTP. HTML

No incio o html era utilizados para a vizualização de artigos e por isso apenas o HTML consegue estruturar como um artigos.

O HTML siginifica Hyper Text Markup Language, Linguagem de Marcação de Hipertexto. Os elementos do HTML informa ao navegador como exibir o conteúdo. Como por exemplo o elemento p que informa isso é um parágrafo, ou o h1 que define um cabeçalho grande.


Parte da configuração, e metadados dentro do head, tudo que é visivel estara dentro do body. Este é um exemplo basico de como funciona o HTML: 

                        <html>
                        <head> 
                            <title> Titulo do Documento </title>
                        </head>
                        <body> 
                            <h1> Este é um cabeçalho </h1>
                            <p> Este é um paragrafo </p>
                        </body>
                        </html>


    Elementos mais comuns:
  • <!DOCTYPE> html que é a declaração de um domumento HTML5
  • <head> elemento da meta infotmações sobre a página
  • <body> elemenento que define o corpo do documento, todo conteúdo visível
  • <title> elemento mostra o título da página HTML
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> elementos de cabeçalho de diferentes tamanho
  • <p> elemento que define um parágrafo

para mais informações sobre HTML consultar: w3scools



CSS

O css é uma ferramenta na qual consegue-se definir estilos para sua pagina. Com ele se faz formatação de fonte, imagens, bordas, e até mesmo animações. Ele garante também o desgner responsivo para que seja utilizado em smartphones.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags <style></style>. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos

Este é um exemplo de css: 

                        body{
                            background-color: lightblue;
                        }

                        h1{
                            color: white;
                            text-align: center;
                        }

                        p{
                            font-family: verdana;
                            font-size:20px;
                        }

para mais informações sobre CSS consultar: w3scools



JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

JavaScript permite a criação de paginas dinamicas com a interação do usuario. E pode ser usada nos navgadores ou diretamente no sistema operacional(NodeJS).

Assim como o Css o JavaScript pode ser escrito diretamente na pagina HTML com o elemenento <script></script>, ou também criar um arquivo separadamente com o a extenção .js com todo o codigo e depois importar no HTML. 

                        <!DOCTYPE>
                        <html>
                        <head>
                            <title> Titulo do Documento </title>
                            <script>
                                function myFunction(){
                                    document.getElementById("demo").innerHTML = "parágrafo Alterado.";
                                }
                            <script>
                        </head>
                        <body> 
                            <h1> Este é um cabeçalho </h1>
                            <p id="demo"> Um parágrafo </p>
                            <button type="button" onclick="myFunction()">Tente</button>
                        </body>
                        </html>

para mais informações sobre JavaScript consultar: w3scools
