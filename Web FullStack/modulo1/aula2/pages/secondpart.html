Ele serve para definir toda estrutura da pagina, a organização,a semântica do documento e a indexação. E funciona em conjunto com o CSS para dar aparencia aos elementos como estilos, fontes, e cores. E com o JavaScript para deixa a pagina dinamicas, como por enviar conversas pelo chat, notificações em sites.


O HTML5 é um padrão aberto, criado por dois grupos:

  • W3C - Word Wide Web Consortium
    • W3c
  • WHATWG - Web Hipertext Application Technology Working Group
    • WHATWG

O W3C é o orgão que cuida das tecnologias Web, o HTML, o CSS, o JavaScript e os padrões dos browser. Antigamente alguns browser como o internet explorer 6 da Microsoft, não seguiam os padrões do W3C. Por conta disto dificultava muito a programação para browsers desse estilos, era preciso programar uma pagina para esses browsers que não seguiam o padrão e uma pagina para as paginas que seguiam o padrão.

O WHATWG é um grupo de trabalho de pessoas interessadas na evolução do HTML e as tecnologias ligadas a tal. Diferente do independente World Wide Web Consortium (W3C) o WHATWG é mantido por pessoas ligadas a entidades e empresas como a Mozilla Foundation, Opera Software ASA e Apple Inc.

Mais os dois juntos que define como será o HTML, e acompanhando os sites das duas organizações, pode se ter ideia, de quais serão as novas tecnologias dos proximos HTML.



Os principios do HTMl, é a não utilização de plugins pesados para exibir conteudo de paginas, e sim utilize o HTML junto com CSS e JavaScript, por que? Por conta que estes plugins aumentam a vunerabilidade da sua aplicação.

Um fator chave do HTML5 é que ele tem que ser multi plataforma, por conta do alto numero de acessos via smartphones, tables, e até mesmo smart TV.

Historia

Em 1989 a melhor forma de compartilhar informações era atraves de e-mail, grupos de noticias (newsgroup), e servidores ftp. Acessava atraves de um desse diversos documentos, a maoria desses documentos eram arquivos de textos, artigos. Mais para cada um desses protocolos tinha um Software especifico, oque dificultava mais ainda.

Tim Berners-Lee propôs a ideia de Hiper textos, que era uma solução melhor, ele foi o idealizador do HTTP e também pelo HTML, na qual a linguagem de marcação para compartilhar os documentos usando formatação, com links, graficos, diagramas.

Antigamente era comum termos paginas como se fossem listas telefonicas, mais com links direcionando para outras paginas que tratavam de algum assunto especifico.

No final do ano de 1989, ele propôs também um piloto de uma projeto, um lista de tags do html, que esta disponivel no link: http://www.3.org/History/19921103-hypertext/hypertext/WWW/MarkUp/Tags.html A proposta era um protocolo cliente/servidor para documentos hipertextos.

O arquivo inicial possuia 18 tagspara definição e formatação de documentos. Neste documento contem as especificações de como seria, o link para o documento é: http://www.w3.org/MarkUp/draft-ietf-iiir-html-01.txt

Em 19991 Berners-Lee publicou o primeiro Web browser, e em 1993 publicou uma biblioteca protável em C chamada de libwww, que permitia o parcer o a renderização das paginas usando HTML. Logo apos lançar o HTML.


Devido essa liberação da criação de browsers, teve a guerra de browsers, mesmo a grande maioria utlzar o internet explorer. Apos o netscape não aguentar e fechar, eles liberaram o codigo fonte do todo navegador, fazendo com que surgise outros grandes navegadores como mozila, mozila firefox, google chrome, depois google investir no motor v8, que é o motor JavaScript.

O HTML é uma linguagem bem permissiva, pode se ver isso como por exemplo na tag de negrito, que caso não feche ela ela apenas irar deixa tudo negrito apartir do momento que abriu, mais não quebrara a pagina e deixara de exibir ela. No inicio o HTML era simplesmente para mostrar hipertextos para os usuario, como por exemplo os artigos cientificos, com as formatações deste esilos. Mais com o tempo surgiu a Web 2.0, que veio falar que a internet é muito poderosa, e por que não utilizar essa rede para fazer sites com as funcionalidades de sistemas, sem a nescesidade de intalação.

Estrutura do HTML5

O HTML é uma linguagem de marcação, e não uma linguagem de programação. Em uma linguagem de programação é possivel criar programas complexos, que tomem decisões, já no html não é possivel fazer isto.


Os elementos do HTML são definos por Tag, a estrutura é definida por abertura e fechamento das Tags, assim <Tag> </Tag> e temos as Tag vazias que abre e fecha, assim <TagVazia/>,usada quando a Tag não tem conteudo, então nas tag temos os tipos: 

                        
    
                            
  • Abertura de Tag: <Tag>
    • 
                            
  • Fechamento de Tag: </Tag>
    • 
                            
  • Tag Vazia: <TagVazia/>
    • 
                            
  • Comentários: <!-- comentário  -->
    •

Apesar do HTML ser uma linguagem muito permissiva como dito anteriormente, como a maoria das linguagem existem as suas boas praticas, para que seu codigo seja mais legivel. Essas boas praticas não inflenciam no funcionamento, apenas faz com que seu documento seja bem formavel, que é um documento que siga a sintaxe correta, e as boas praticas de organização.

Algumas boas praticas são:

  • Todas as tag são sintaticamente corretas
    • ou seja toda tag aberta em letra minucula assim <p> deve ser fecada em minucula, assim </p>, o mesmo se for maiuscula.
  • Todas as tags abertas precisam ser fechadas
    • podendo ser tags vazias que já fecham exemplo <TagVazia/>, mais todas as outras precisam ser <p>Olá Mundo</p>.
  • Todas os atributos devem ser delimitados com aspas
    • como o exemplo a imagem definida com um src, que é o caminho dela deve ser <img src="">
  • As tags devem ser fechadas na ordem de sua abertura:
    • por exemplo o negrito que pode ser aberto asim usando o <b>, mais não é uma boa pratica usar assim <b><i></b></i>, mais a boa pratica seria <b><i></i></b>



A seguir temos a estrutura basica para ter um arquivo HTML5: 

                        <!DOCTYPE html>
                        <html lang="pt">
                        <head> 
                            <meta charset="UTF-8">
                            <title> Titulo do Documento </title>
                        </head>
                        <body> 
                           Conteudo o site.
                        </body>
                        </html>

No começo do documento definos o formato do nosso documento de marcação usando o <!DOCTYPE html>, ou colocando o <!DOCTYPE xml>, ou outro formato de marcação.

Na Tag html temos o lang como atributo para sinalizar o lingua do site, os pacotes de lingua não estão relacionados com o html, e sim com o navegador. O navegador não ira traduzir, so ira identificar que esta em portugues, ou qual qual lingua estiver.

Dentro no head colocamos a codificação de caracteres utilizando o <meta charset="UTF-8">, por que não será sempre correto, por vaziações de editor de texto, ou sistema operacional, ou como salva, as vezes saira sem pontuação, ou com pontos de interogação.

Alem disto podemos adicionar mais coisas, para faciliar a indexação e o usuario. Como por exemplo a acessibilidade, tipo um usuario com deficiencia visual, que ira precisar de um leitor de tela, que podera ler esses meta dados um deles é o charset, mais pode se criar meta datos expecificos, como o keywords, que destaca as palavras chaves na indexação. 

                        <!DOCTYPE html>
                        <html lang="pt">
                        <head> 
                            <meta charset="UTF-8">
                            <meta charset="keywords" content="curso html uenp"/>
                            <title> Titulo do Documento </title>
                        </head>
                        <body> 
                           Conteudo o site.
                        </body>
                        </html>


    Elementos mais comuns:
  • <!DOCTYPE> html que é a declaração de um domumento HTML5
  • <head> elemento da meta infotmações sobre a página
  • <body> elemenento que define o corpo do documento, todo conteúdo visível
  • <title> elemento mostra o título da página HTML
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> elementos de cabeçalho de diferentes tamanho
  • <p> elemento que define um parágrafo
