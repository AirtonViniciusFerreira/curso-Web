Aula 2 - CSS

Agora vamos ver como organizar os documentos e deixá-los mais bonitos. O CSS (Cascading Style Sheets) define os padrões de formatação do documento, a tipografia, as fontes, tamanho dos elementos, imagens, os layouts e até mesmo animações.



História

O CSS não veio junto com o HTML, depois de um tempo usando a formatação do HTML, surgiu a necessidade de especificação de fazer isto de forma separada, para que possa ter o reuso, a organização e a facilidade de construção, como por exemplo, para mudar todos elementos "h1" de uma página era necessário ir um por um, e mudar com o CSS é possivel mudar todos de uma só vez. Então em 1996 o CSS level 1 for publicado.

Com as guerras dos browsers, no início tinha browser que implementava essa especificação do CSS e alguns não implementava, e alguns eram diferentes e tinham algumas mudanças, e até hoje o CSS é um grande problema, com algumas especificações como degrade que é nova e tem estruturação específica para o mozila e outra para o chrome.

Antigamente as formatações precisavam estar em todas as tags que desejava, por não ter uma especificação do tipo classe que tem no CSS. As formatações eram assim: 

    <font face="verdan>Isso é algum texto</font>
    <center>O texto será alinhado ao centro</center>
    <html bgcolor=#f43a55 color#000000> </html>


Interação CSS e HTML

A interação acontece através de estilos que se coloca no HTML. Há três maneiras de colocar esse estilos, sendo: colocando o atributo style dentro da tag que deseja por o estilo e coloca um estilo apenas; caso tenha mais de um estilo coloca a tag style no documento lá em cima na dentro da tag head <style> aqui vão os estilos </style>; ou se não usar a tag link para referenciar outro documento com extensão .css para a reutilização desses estilos a tag é a <link rel="stylesheet" href="style.css">.

Sintaxe

O seletor é um conjunto de regras que termina a formatação do elemento, temos o seletor que usa na tag style e no arquivo de estilo.css, e temos o seletor do atributo style diretamente na tag, porém não é recomendado no caso de mais de um estilo ou mais de uma tag igual.

Este é o seletor da tag style ou no arquivo, e como podemos ver usa o ponto e vírgula para ir a para próxima instrução como é usado normalmente em desenvovimentos. E pode ser reduzido tudo a uma linha também como mostrado na segunda forma, mas para facilitar a leitura usa a primeira forma. 

    
    seletor{
        regra1;
        regra2;
        ...
        regraN;
    }

    seletor{ regra1; regra2; ... regraN; }

Já no atributo da tag usando o style=" ", para atribuir os estilos, normalmente usado para casos no qual é apenas um estilo, e se for apenas para aquela tag em especifico. Mas também funcionará caso coloque mais de um estilo como no segundo exemplo: 

    <tag style="regra1;">
    <tag style="regra1; rega2; ... regraN">

O seletores de elementos define a formatação para todos os elementos daquele tipo, por tanto se colcoar um seletor "h1", "p", "section" e "pre", ele formatará todos os elementos para aquele determinado estilo, como nos exemplos a seguir que muda todos h1 e todos p: 

    h1{color: blue; font-size:12px}

    p{
        text-align: center;
        color: blue;
        font-size: 16px;
    }

Temos o seletor do ID's onde eu crio uma tag, e coloco um nome para eles, uma regra de um documento bem formado, não pode conter mais de uma tag com o mesmo nome, mas não é obrigado a identificar todas as tags, mas se quiser identificar poderá colocar um estilo para aquela tag com quele nome.

Para usar o seletor do ID basta colocar #botaoEnviar{regra1;}, o hashtag (#) indica que aquele seletor é referenciado ao ID que está na frente do #.Para colocar o id basta inserir o atributo id="nome da tag" dentro da tag, assim <tag id="botaoEnviar">.Então ficaria assim: 

    <style>
        #botaoEnviar{
            color: red;
        }
    </style>
    <button id="botaoEnviar">Enviar</button>

Outro seletor que é um dos mais utilizados hoje em dia é o seletor de classes, para criar uma classe basta colocar o ponto (.) antes do nome da classe assim .nomeClasse{regra1; regra2;}, a classe nada mais é do que um apelido para a tag, para usá-la na tag, utiliza-se o atributo class="" dentro da tag desejada. Como no exemplo: 

    <style>
        .error{
        color: red;
        }
    </style>
    <h1 class="error">ocorreu um erro<h1>
    <span class="error">
        o erro deve ser corrigido<span>

Diferente do ID a classe pode ser utilizada várias vezes no documento, então pode ser reutilizada em diversas tags e podemos colocar até mais de uma classe caso o elemento possua estilos de uma ou mais classes.

O box model temos os esquemas de caixa, tendo altura e largura, o width é a largura, e o height é a altura da caixa. A caixa tem os elementos padding, que é a distância do conteúdo até a borda. Border é a borda da caixa, que fica entre o padding e a margin. Margin é a margem que é a última camada da caixa que fica depois da borda. Todos esses elementos, padding, border, margin terão todos os lados, direita (rigth), esquerda (left), cima (top), baixo (bottom), e pode ser acessados separadamente.

Para acessar esses elementos basta colocar no style o elemento-direção, como por exemplo margin-top: 10px, fará com que apenas a margem de cima tenha 10 pixel, caso queira deixar todos iguais, coloque apenas o elemento, assim margin 10px, agora todos os lados da margem tem 10 pixel.



Unidades no CSS

A unidade de medida é uma das dificudades em css, por conta da diferença das resoluções da tela, podem ter alterações enormes caso use a unidade de centímetro, ela é usada normalmente para papel, formatação de documentos impressos, assim teria uma precissão maior. Mas para as resoluções da tela utilizamos em pixel ou porcentagem.

Temos duas categorias de unidades: as unidades Absolutas e as unidades Relativas.

As unidades Absolutas, são as unidades que usamos mais no dia a dia, como centímetro, milímetro, polegada, pixel, ponto, paica. Como mostramos no exemlo.

UnidadeDescrição
cmCentímetro
mmMilímetro
inPolegada

pcPaica (4,23mm)

As unidades relativas estão relacionadas com tamanho. Temos a "porcentagem", o "em" que está relacionada com a fonte que foi escolhida pegando a letra M da fonte escolhida, e o "vh".
