Introdução ao Desenvolvimento Web FullStack-CCT

Curso de FullStack-CCT

Neste curso iremos aprender sobre as tecnologias para aprender a desenvolver um site, iniciaremos com o front-end, com as tecnologias de HTML, CSS e JavaScript, e posteriormente veremos Git e GitHub.

Também teremos Análise de requisitos, Design thinking, IHC, Usabilidade Interfaces Responsivas e Mobile Fist, MaterializeCSS.

Também teremos a parte de back-end utilizando a linguagem de programação php, além disso apresentaremos a construção de uma API Rest e usaremos o Larevel, com o MVC, mostraremos boas práticas em todas as tecnologias, diferenças entre programador back-end, front-end e fullStack.

Este curso ainda está em desenvolvimento, mas podem ser acessadas algumas aulas, que já estão prontas, temos alguns resumos por enquanto e posteriormente, posteremos o restante, e alguns conteúdos extras. A princípio temos estas páginas, as Aulas 1, 2 e 3 do módulo 1, e a prototipagem. Bom estudos.
